# AI Usage

## Tool

Claude (claude-sonnet-4-6) by Anthropic — digunakan melalui Claude Code CLI.

## Penggunaan

AI digunakan sebagai asisten kode dalam proses pengerjaan proyek ini.

### Backend (Laravel)
- Scaffolding struktur controller, form request, dan routing API
- Implementasi query Eloquent, relasi model, dan validasi input
- Penulisan PHPUnit feature test

### Frontend (Vue 3 + TypeScript)
- Setup komponen Vue 3 dengan TypeScript strict
- Konfigurasi Axios interceptor, Pinia store, dan Vue Router
- Implementasi kanban drag & drop dengan optimistic update
- Penulisan Vitest component test

### Tooling & Konfigurasi
- Konfigurasi Docker Compose, Dockerfile backend dan frontend
- Setup CORS, phpunit.xml, dan environment
